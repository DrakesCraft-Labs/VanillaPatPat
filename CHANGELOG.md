# Changelog

## [1.0.0] - 2026-04-20

### Added
- Initial release of VanillaPatPat.
- Patting mechanic: Shift + right-click on any living entity with an empty hand.
- Heart particles spawned above the entity's head.
- Action bar message on successful pat.
- 500ms cooldown to prevent spam.
- Cat purr sound on pat.
